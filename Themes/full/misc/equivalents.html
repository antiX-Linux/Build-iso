Equivalent Applications

Current antiX-Full-Stable v Windows (Mainly XP Home)


In antiX you are able to do most of the day-to-day things you do in Windows.  Because antiX is not Windows, the apps you use to carry out your daily activities are not the same as the ones used in Windows. 
  • In some cases antiX apps extend the range of things you can do
  • In lots of cases the way the antiX apps are used is quite familiar
  • In almost every case the name of the antiX apps are different 
The following table is an introduction to antiX apps.  It shows some sections of the antiX menu and lists the apps in each of them.  It also gives a very brief description of what each app does and the name of an equivalent app often found in Windows.

Rather than offering a single a single monolithic app that performs a range of tasks, the antiX philosophy is to offer multiple small apps.  One advantage of this approach is wider user choice.   Sometimes antiX provides more than one app that perform similar roles.  In these cases an extra choice is built in. 

For some apps, there is an introductory video available.  To watch it simply click the icon in a fully featured web browser.  In antiX-Full-Stable this is named Iceweasel.

Where a video is not available it is still often possible to get more information.  Start the app from the main antiX menu and in the app window click the help button.  This usually displays a button named "About".  Clicking this will often show the address of the home web site for the app.
